# Channel Brain — Frontend

Next.js 15 + React 19 + TypeScript + Tailwind.

## Setup

From the project root:

```
cd frontend
npm install
```

Copy the env file:

```
copy .env.local.example .env.local
```

## Run dev server

```
npm run dev
```

Opens on http://localhost:3000

## What Session 3 delivers

- Next.js project runs locally
- Placeholder homepage renders with correct fonts + colors
- Tailwind configured with our palette
